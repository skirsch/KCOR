# US Mortality Analysis Workspace

This directory is for US-focused mortality analysis relevant to the debate question:

**In the US, did the mRNA COVID vaccine likely net save lives or cost lives through the end of 2022?**

## Recommended Starting Point

Start with **UCOD / underlying-cause-of-death style national mortality data** first.

Why:

- it provides the cleanest national mortality backbone
- it supports sex and age stratification better than many ad hoc summaries
- it helps establish whether there are unusual male/female or age-specific all-cause patterns after rollout
- it gives a baseline before using noisier or more assumption-heavy vaccine-effect narratives

## Suggested Analysis Order

1. Weekly or monthly all-cause mortality by sex and age band over calendar time
2. Compare pre-rollout versus post-rollout patterns
3. Add excess mortality framing where appropriate
4. Compare consistency across multiple US sources
5. Only then move into more causal or vaccine-linked interpretations

## Candidate US Data Sources To Compare

- UCOD / underlying cause of death data
- excess mortality datasets
- CDC / NCHS weekly mortality files
- vaccination uptake by age and sex if available

## Practical Goal

The goal is not to assume one source is decisive.

The goal is to:

- look for consistency across US datasets
- identify where signals are robust versus source-specific
- test whether female versus male patterns changed meaningfully after rollout
