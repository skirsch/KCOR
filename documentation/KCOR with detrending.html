KCOR from Mortality Rates — Concise Guide

Protocol

  1. Compute weekly deaths Dw and person-time PTw (≈ Alive + 0.5·Dead).
  2. Weekly crude mortality rate (CMR): rw = Dw/PTw.
  3. Select cohorts to compare (e.g. Dose 2 vs Dose 0, same birth-year).
  4. Form weekly log-ratio: yw = log(rw,treat/rw,ref).
  5. Detrend yw by subtracting a baseline fit in a “plateau” window (e.g. calendar 2023) ⇒ ỹw.
  6. Apply weights ww = min(PTw,treat,PTw,ref) (or =1).
  7. Cumulative KCOR: 
    KCOR(t) = exp( Σ ỹ_w·w_w / Σ w_w )
  8. 95% CI: Var(log RRw) ≈ 1/Dw,treat + 1/Dw,ref. Propagate through weighted mean, then exponentiate.
  9. If detrending=off, KCOR(t) = ratio of cumulative CMRs.

Key points

  • Detrending is done on the ratio, not each arm separately.
  • One correction per (BirthYear × Dose_vs_0) curve.
  • If you skip detrending, both formulas collapse to the raw cumulative ratio.
  • KCOR(t) & CIs can be computed each week for plotting.

Pseudocode

for each week w:
    r_treat[w] = D_treat[w]/PT_treat[w]
    r_ref[w]   = D_ref[w]/PT_ref[w]
    y[w] = log(r_treat[w]/r_ref[w])

# Detrend using baseline window
ỹ[w] = y[w] - baseline_fit(y)

# Weighted cumulative mean of detrended log-ratios
mu[t] = Σ(ỹ[:t]*w[:t]) / Σ w[:t]
KCOR[t] = exp(mu[t])

# Variance for CI
var_mu[t] = Σ(w[:t]^2 * var_log_rr[:t]) / (Σ w[:t])^2
SE = sqrt(var_mu[t])
CI = exp(mu[t] ± 1.96·SE)
